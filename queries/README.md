# Saved Queries

This directory contains natural language queries used for infrastructure discovery, organized by category.

## Usage
1. Save queries in appropriate subdirectory
2. Include both query and response
3. Date and tag each query session
4. Reference back to main DISCOVERY.md

## Categories
- `infrastructure/` - General Azure resource queries
- `security/` - NSGs, identity, B2C queries  
- `networking/` - VNets, subnets, load balancers
- `kubernetes/` - AKS-specific queries
